# 05 — Loops & Flow Control

Loops help us repeat a block of code until a condition is met.

### Covered
- while loop
- for loop
- do-while loop
- break & continue
- basic flow control

### Examples include
- Print numbers (1–10, 1–n)
- Sum of first n natural numbers
- Square pattern printing
- Reverse a number
- Prime number check

All programs are simple, commented, and beginner-friendly.
