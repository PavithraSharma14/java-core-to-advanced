# 🧪 Sample Code

### Check Even or Odd (Bitwise)

```java
public static boolean isEven(int num) {
    return (num & 1) == 0;
}
```

### Get i-th Bit

```java
public static int getIthBit(int num, int i) {
    return (num >> i) & 1;
}
```
