# Practice Questions - Inheritance

## 🔹 Basic
1. Create a class `Vehicle` and extend it with `Car`.
2. Show how child class can access parent class methods.

## 🔹 Intermediate
3. Create a multilevel inheritance example with `Person → Employee → Manager`.
4. Demonstrate hierarchical inheritance using `Shape → Circle, Rectangle`.

## 🔹 Advanced (IMPORTANT)
5. Write a program showing method overriding with runtime polymorphism.
6. Create a banking system using inheritance (Account → Savings, Current).

## 🔹 Interview Questions
7. What is the difference between overloading and overriding?
8. Why Java does not support multiple inheritance with classes?
9. What is runtime polymorphism?

## 🎯 Goal
Solve these without looking at solutions.
