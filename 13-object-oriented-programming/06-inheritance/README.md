# Inheritance in Java

This module demonstrates the concept of inheritance in Java using structured examples.

## 📌 Topics Covered
- Single Inheritance
- Multilevel Inheritance
- Hierarchical Inheritance
- Method Overriding (Runtime Polymorphism)

- examples/ → Contains code demonstrations
- practice-questions.md → Interview-focused problems

- Understand how classes reuse properties using inheritance
- Learn method overriding and runtime polymorphism
- Apply OOP principles in structured programs
