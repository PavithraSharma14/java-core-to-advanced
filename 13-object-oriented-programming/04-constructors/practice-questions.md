# Practice Questions — Constructors

1. Create a `Student` class with a default constructor that prints "Student created".

2. Create a `Car` class with a parameterized constructor that sets `brand` and `year`.

3. Create a `Rectangle` class with a constructor that initializes `length` and `width`.

4. Create two constructors in a `Book` class:
   - one without parameters
   - one with title and author.

5. Create a class `Employee` with constructor overloading:
   - one constructor with `name`
   - one constructor with `name` and `salary`.
