# Practice Questions - Polymorphism

## 🔹 Basic
1. Create a class with overloaded methods for addition.
2. Show method overloading using different data types.

## 🔹 Intermediate
3. Create a parent class `Shape` and override method `draw()` in child classes.
4. Demonstrate runtime polymorphism using Animal → Dog, Cat.

## 🔹 Advanced
5. Build a payment system using polymorphism (Payment → UPI, Card, Cash).
6. Show difference between overloading and overriding using code.

## 🔹 Interview Questions
7. What is polymorphism in Java?
8. Difference between compile-time and runtime polymorphism?
9. Can we override static methods? Why?

## 🎯 Goal
Be able to explain with examples without memorizing.
