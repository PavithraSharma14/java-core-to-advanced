# Practice Questions — Encapsulation

1. Create a `Student` class with private fields `name` and `age`.
   Provide getter and setter methods.

2. Create a `BankAccount` class with a private `balance`.
   Add methods to deposit and check balance.

3. Create an `Employee` class with private `salary`.
   Ensure salary cannot be negative.

4. Create a `Product` class with private `price`.
   Add methods to update and retrieve the price.

5. Create a `Car` class with private `speed`.
   Ensure speed cannot exceed a certain limit.
