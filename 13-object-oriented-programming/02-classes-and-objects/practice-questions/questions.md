# Practice Questions — Classes and Objects

## Basic Level

1. Create a `Student` class with fields `name` and `age`.  
   Create an object and print the student details.

2. Create a `Book` class with fields `title` and `author`.  
   Create an object and display the book information.

3. Create a `Car` class with fields `brand` and `year`.  
   Create an object and print its details.

4. Create a `Laptop` class with fields `brand` and `price`.  
   Print the laptop information using an object.

---

## Intermediate Level

5. Create a `Rectangle` class with `length` and `width`.  
   Add a method to calculate and print the area.

6. Create a `Circle` class with a `radius` field.  
   Add a method to calculate the area of the circle.

7. Create a `BankAccount` class with a `balance` field.  
   Add a method to deposit money and display the balance.

8. Create a `MobilePhone` class with fields `model` and `price`.  
   Add a method that prints the phone details.

---

## Challenge Level

9. Create two objects of a `Student` class and print their details.

10. Create an `Employee` class with fields `id`, `name`, and `salary`.  
    Display the employee information using a method.

11. Create a `Product` class with `name` and `price`.  
    Create multiple objects and print their details.

12. Create a `Movie` class with fields `title`, `director`, and `rating`.  
    Print the movie details using an object.
