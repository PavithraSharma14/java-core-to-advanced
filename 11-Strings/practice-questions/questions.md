# String Practice Questions

These questions are meant for hands-on practice after completing string basics and problems.
---

## Q1. Reverse a String

Given a string, reverse it without using any built-in reverse method.

**Hint:**
- Use `charAt()`
- Loop from end to start

---

## Q2. Count Vowels in a String

Count the number of vowels (`a, e, i, o, u`) in a given string.

**Hint:**
- Convert characters to lowercase
- Check using condition

---

## Q3. Check Anagram

Given two strings, check whether they are anagrams.

Example: 
listen → silent (true)
hello → world (false)

**Hint:**
- Convert to lowercase
- Use frequency array of size 26

---

## Q4. Toggle Case of Characters

Convert:
- uppercase → lowercase
- lowercase → uppercase

Example:
Input: "JaVa"
Output: "jAvA"

**Hint:**
- ASCII values
- `char + 32` or `char - 32`
